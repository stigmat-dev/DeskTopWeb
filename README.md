# АИС «ИНФО-ПРО»

Автоматизированная информационная система «ИНФО-ПРО» включает в себя две подсистемы:

1. АСПОЗ - Автоматизированная система подачи и обработки заявок
2. АСУП - Автоматизированная система учета пациентов
